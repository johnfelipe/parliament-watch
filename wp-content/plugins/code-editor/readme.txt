== Description ==

Brings syntax highlighting and line numbers to the theme- and plugin-editor of wordpress. Supports php, css, html and js.

Visit the [Plugin Homepage](http://www.naden.de/blog/wordpress-code-editor "Wordpress Code Editor") for more details.

== Installation ==

1. Unpack the zipfile code-editor-X.y.zip
1. Upload folder `code-editor` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does the plugin supports c++ ? =

No, because it's not very likely, that you'll be in need of coding c++ via the wordpress admin panel.
